# start-default-nodejs

## About the app:

It is a simple project for start node.js projects using MVC Pattern.

## Technologies and Packages:

- Node.JS
- Express
- Cors
- Dotenv
- Nodemon

## How to install:

```
1 - npm install
2 - Rename .env.example to .env and set your stytch config ( project_id and secret ).
2 - yarn/npm start
```
